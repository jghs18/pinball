****Information****
*In order to change the color of the main ball press the 'space' key
*In order to move the ball up press the 'w' key
*In order to move the ball down press the 's' key
*In order to move the ball right press the 'd' key
*In order to move the ball left press the 'a' key
*To stop the ball press the key that would move the ball in the opposite direction
*To speed up the ball repeatedly press the key you want the direction to move
*To add a new ball to the frame press the 'n' key
**The main ball will react with the other balls by bouncing off of them, however in some cases the balls
**may be traveling to fast for the computer to check if they touch so they will not bounch. I will try to 
**address this issue in the future
*Also note that in my world rules of physics don't always apply so deal with it :)
